# Knutt vs. Herzmuschel (Web)

Vue 3 + TypeScript + Vite + Pinia + Chart.js.

## Befehle

| Befehl        | Beschreibung              |
| ------------- | ------------------------- |
| `npm run dev` | Entwicklungsserver        |
| `npm run build` | Produktionsbuild (`dist/`) |
| `npm run preview` | Build lokal testen      |
| `npm run test` | Vitest (Spiellogik)      |
| `npm run lint` | ESLint                   |

Grafiken liegen unter `public/assets/` (Kopie aus `../media/`).

Weitere Infos: [README im Repo-Root](../README.md).
